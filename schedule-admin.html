Language settings

Users can pick up to 2 languages in the app. Enable the languages you have translations for. Disabled languages will not appear in the user's dropdown.

Language selection is limited to 2 per user session. If a user picks the same language twice, the app automatically prevents duplicates.

How Gumroad License Keys work

Setup guide
01

Create products

Create 2 products on Gumroad — one for ₹199 (single) and one for ₹499 (all). Enable License Keys in each product's settings.

02

Add permalinks

Copy the product permalink from Gumroad and paste it into the two fields below. The app uses this to verify keys against the right product.

03

User buys

User clicks Buy in the app → Gumroad checkout → payment → Gumroad emails them a unique license key automatically.

04

User enters key

User pastes their key into the app. The app calls Gumroad API to verify → unlocks categories → saves to browser localStorage.
